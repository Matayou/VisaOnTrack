# UI/UX Specialist Skill

**Purpose:** Expert UI/UX designer for SawadeePass with focus on consistency, beauty, and conversion optimization.

## How to Use

### Via Skill Command
```
/skill ui-ux-specialist
```

### What It Does
- Designs beautiful, consistent pages and components
- Applies SawadeePass design system standards
- Optimizes for conversion (CRO)
- Ensures accessibility (WCAG 2.1 AA+)
- Provides detailed implementation guidance

### When to Use
- Creating new pages or flows
- Building new UI components
- Reviewing/improving existing UI
- Optimizing conversion funnels
- Solving accessibility issues
- Establishing design patterns

## What You'll Get

1. **Component Code** - Production-ready React/TypeScript
2. **Design Rationale** - Why these decisions?
3. **CRO Notes** - Conversion optimization insights
4. **Accessibility Notes** - ARIA, keyboard, screen reader support
5. **Responsive Behavior** - Mobile, tablet, desktop
6. **Variants** - Size, color, state options

## Design Standards Applied

✅ Trust-first design principles
✅ Tailwind + shadcn/ui patterns
✅ Phase 1 & 2 design system (100% coverage)
✅ Unified components (Header, Footer, Input, FormField)
✅ Standardized utilities (error-handling, loading-messages)
✅ Mobile-first responsive design
✅ WCAG 2.1 AA accessibility
✅ CRO best practices

## Example Request

> "Design a provider profile card that shows credentials, rating, services, and an 'Unlock Details' CTA. Optimize for trust and conversion."

## Quality Bar

Minimum score to ship: **85/100**
- Visual Design: 25%
- User Experience: 30%
- Conversion Optimization: 20%
- Accessibility: 15%
- Technical Quality: 10%
