# Compact Sizing Guidelines

## Size Reductions Applied

**Font Sizes:**
- H1: `text-4xl` → `text-2xl` (36px → 24px)
- H2: `text-2xl` → `text-xl` (24px → 20px)
- H3: `text-xl` → `text-base` (20px → 16px)
- Body: `text-base` → `text-sm` (16px → 14px)
- Small: `text-sm` → `text-xs` (14px → 12px)

**Icons:**
- Large: `w-8 h-8` → `w-6 h-6`
- Medium: `w-6 h-6` → `w-5 h-5`
- Small: `w-5 h-5` → `w-4 h-4`

**Padding:**
- Cards: `p-10` → `p-6`
- Sections: `mb-10` → `mb-6`
- Form fields: `py-3.5` → `py-2.5`
- Buttons: `py-3.5 px-6` → `py-2 px-4`

**Spacing:**
- Gaps: `gap-6` → `gap-4`
- Margins: `mb-8` → `mb-6`
- Forms: `space-y-6` → `space-y-4`

**Rounded Corners:**
- Cards: `rounded-2xl` → `rounded-xl`
- Buttons: `rounded-xl` → `rounded-lg`
- Inputs: `rounded-xl` → `rounded-lg`
